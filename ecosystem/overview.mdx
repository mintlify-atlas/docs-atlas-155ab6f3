The Fuse ecosystem is a growing collection of decentralized applications, infrastructure providers, developer tools, and community members working together on the Fuse Network. As an open-source, EVM-compatible blockchain, Fuse supports anyone—from individual developers to large organizations—who wants to build or use Web3 applications.

## What makes up the Fuse ecosystem

The ecosystem spans several categories of projects and participants.

**DeFi** — Decentralized exchanges, lending protocols, yield optimizers, and stablecoin infrastructure are all active on Fuse. Voltage Finance is the primary DEX, providing liquidity and token swap services.

**NFTs** — NFT marketplaces, minting tools, and gaming projects use Fuse for low-cost, fast-finality transactions that make on-chain asset ownership practical.

**Wallets** — From browser extensions like MetaMask to mobile-native apps like Volt, users have a wide choice of wallets that support the Fuse Network. Smart contract wallets are available through FuseBox.

**Developer infrastructure** — RPC providers, block explorers, oracle services, and indexing tools form the foundation that application developers build on. See [Web3 developer tools](/ecosystem/web3-tools) for a full list.

**Validators and governance** — Validators maintain network security and process transactions. FUSE token holders can stake or delegate to validators and vote on Fuse Improvement Proposals (FIPs).

## Fuse Explorer

[Fuse Explorer](https://explorer.fuse.io) is the official block explorer for Fuse Network, built on the open-source Blockscout platform. Use it to search blocks, transactions, and addresses, verify smart contracts, and browse token transfers.

The testnet explorer for Fuse Spark is available at [https://explorer.fusespark.io](https://explorer.fusespark.io).

**Blockscout Pro API** — A premium tier of the Explorer API offers higher rate limits, multichain support through unified chain ID endpoints, and a free tier. Get started at the [Blockscout Developer Portal](https://dev.blockscout.com/).
